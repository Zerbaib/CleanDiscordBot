
### **`/botinfo`**:

    ```

    ```

    ```
    Show info about the server
    ```

### **`/userinfo {user}`**:

    ```
    Show info about you or other user
    ```

### **`/ping`**:

    ```
    Show the ping of the bot
    ```

### **`/poll <text>`**:

    ```

    ```
